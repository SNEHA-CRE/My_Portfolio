# React My Portfolio

A responsive React portfolio website for Sneha Kharat, featuring experience, skills, projects, certifications, education, and contact details.

## Run locally

1. Install dependencies:

   npm install

2. Start the development server:

   npm run dev

3. Open the displayed local URL in your browser.

## Build

npm run build

## Notes

- Built with Vite and React.
- Includes a clean UI with responsive layout.
- Update `src/App.jsx` to personalize text, projects, or contact links.
